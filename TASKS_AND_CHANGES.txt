Implementation Notes
- The analyzer uses heuristic parsing to work on incomplete or poorly formatted resumes.
- It avoids unsupported hallucinations by returning null or cautious summaries where details are missing.
- ATS scoring is weighted across formatting, keyword usage, skills, projects, and experience.
- Job matching is optional and activates only when a job description is provided.
- Verification was completed with a sample resume using the local Python runtime.
- The UI runs fully locally in the browser and does not require an external service.
- PDF parsing is optional because the local environment may not have `pypdf` installed yet.
